Introduction to Java

What is Java?

Java is a powerful, platform-independent programming language used for:

  • Mobile Applications (Android)
  • Backend Services & APIs
  • Desktop Applications
  • Web Applications
  • Enterprise Software

Why Java?

  • Platform Independent: Write Once, Run Anywhere (WORA)
  • Object-Oriented: Clean, modular code structure
  • Robust & Secure: Memory management and strong type checking

Key Points:

  • Java code compiles to bytecode, not machine code
  • JVM (Java Virtual Machine) executes bytecode on any platform
  • One .java file → One .class file after compilation
  • Platform independence makes Java ideal for cross-platform applications

First Java Program - Hello World

Program Structure

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Understanding the Code:

  • public class HelloWorld - Declares a public class
  • public static void main(String[] args) - Entry point of the program
  • System.out.println() - Prints output to console
  • ; - Every statement ends with a semicolon

Key Points:

  • Class name must match the filename (HelloWorld.java)
  • main() method is the starting point of execution
  • Java is case-sensitive - Main ≠ main
  • All code must be inside a class

Compilation & Execution

// Compile the program
javac HelloWorld.java

// Run the program
java HelloWorld

// Output:
Hello, World!

Example: Simple Calculator

import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        System.out.print("Enter first number: ");
        int num1 = input.nextInt();
        
        System.out.print("Enter second number: ");
        int num2 = input.nextInt();
        
        int sum = num1 + num2;
        int difference = num1 - num2;
        int product = num1 * num2;
        double quotient = (double) num1 / num2;
        
        System.out.println("Sum: " + sum);
        System.out.println("Difference: " + difference);
        System.out.println("Product: " + product);
        System.out.println("Quotient: " + quotient);
        
                        input.close();
    }
}

Key Points:

  • Scanner is used to read user input from console
  • nextInt(), nextFloat(), nextLine() for different types
  • Always close Scanner after use to prevent memory leaks
  • Use type casting like (double) to avoid integer division

Example: Even or Odd

import java.util.Scanner;

public class EvenOdd {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Enter a number: ");
        int number = sc.nextInt();
        
        if (number % 2 == 0) {
            System.out.println(number + " is Even");
        } else {
            System.out.println(number + " is Odd");
        }
        
        sc.close();
    }
}

Key Points:

  • if-else executes code based on boolean conditions
  • % (modulo) operator gives the remainder of division
  • Use else if for multiple conditions
  • switch is cleaner for checking one variable against multiple values

Switch Statement Example

import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Enter operator (+,-,*,/): ");
        char operator = sc.next().charAt(0);
        
        System.out.print("Enter two numbers: ");
        int a = sc.nextInt();
        int b = sc.nextInt();
        
        switch (operator) {
            case '+':
                System.out.println("Result: " + (a + b));
                break;
            case '-':
                System.out.println("Result: " + (a - b));
                break;
            case '*':
                System.out.println("Result: " + (a * b));
                break;
            case '/':
                System.out.println("Result: " + (a / b));
                break;
            default:
                System.out.println("Invalid operator");
        }
        
        sc.close();
    }
}

Loop Types

For Loop

Use when you know the number of iterations

for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
While Loop

Use when condition is checked before execution

int i = 1;
while (i <= 5) {
    System.out.println(i);
    i++;
}
Do-While Loop

Use when body should execute at least once

int i = 1;
do {
    System.out.println(i);
    i++;
} while (i <= 5);

Example: Multiplication Table

import java.util.Scanner;

public class MultiplicationTable {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Enter a number: ");
        int num = sc.nextInt();
        
        System.out.println("Multiplication Table of " + num);

Example: Pattern Printing

public class StarPattern {
    public static void main(String[] args) {
        int rows = 5;
        
        // Right-angled triangle
        for (int i = 1; i <= rows; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.print("* ");
            }
            System.out.println();
        }
        
        // Output:
        // * // * * // * * * // * * * * // * * * * * }
}

Key Points:

  • for loop: Use when you know the number of iterations
  • while loop: Condition checked before execution
  • do-while loop: Executes at least once, condition checked after
  • Nested loops are used for patterns and 2D operations

Example: Factorial Function

import java.util.Scanner;

public class FactorialCalculator {
    
    // Function to calculate factorial
    public static int factorial(int n) {
        if (n == 0 || n == 1) {
            return 1;
        }
        
        int result = 1;
        for (int i = 2; i <= n; i++) {
            result *= i;
        }
        return result;
    }
    
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Enter a number: ");
        int num = sc.nextInt();
        
        int fact = factorial(num);
        System.out.println("Factorial of " + num + " is: " + fact);
        
        sc.close();
    }
}

Example: Prime Number Checker

import java.util.Scanner;

public class PrimeChecker {
    
    // Function to check if number is prime
    public static boolean isPrime(int n) {
        if (n <= 1) {
            return false;
        }
        
        for (int i = 2; i <= Math.sqrt(n); i++) {
            if (n % i == 0) {
                return false;
            }
        }
        return true;
    }
    
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Enter a number: ");
        int num = sc.nextInt();
        
        if (isPrime(num)) {
            System.out.println(num + " is a Prime number");
        } else {
            System.out.println(num + " is not a Prime number");
        }
        
        sc.close();
    }
}

Key Points:

  • Functions promote code reusability and modularity
  • return statement sends value back to the caller
  • void functions don't return any value
  • Use descriptive names and keep functions focused on one task

Mini Challenge: Pattern Party

Create your own creative star or number pattern using loops! Here are some examples to inspire you:

Pyramid Pattern

public class Pyramid {
    public static void main(String[] args) {
        int rows = 5;
        
        for (int i = 1; i <= rows; i++) {
            // Print spaces
            for (int j = 1; j <= rows - i; j++) {
                System.out.print(" ");
            }
            // Print stars
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}

// Output:
//     *
//    ***
//   *****
//  *******
// *********

Diamond Pattern

public class Diamond {
    public static void main(String[] args) {
        int n = 5;
        
        // Upper half
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= n - i; j++) {
                System.out.print(" ");
            }
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            System.out.println();
        }
        
        // Lower half
        for (int i = n - 1; i >= 1; i--) {
            for (int j = 1; j <= n - i; j++) {
                System.out.print(" ");
            }
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}
